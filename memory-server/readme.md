# memory-server

Эта директория про memory-server.

Мы хотим использовать memory-server, чтобы агенты не теряли память между запусками: решения, договоренности, удобные паттерны, ошибки и прочий контекст, который обычно приходится заново объяснять в каждом новом треде.

Здесь будет простая коллекция скиллов, гайдов и заметок для агентов, которые мы нашли удобными при работе с persistent memory.

Не кладем сюда секреты, customer data и прочую радость, которую потом придется выжигать из истории.
